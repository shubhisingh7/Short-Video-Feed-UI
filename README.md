# 🎥 Short Video Feed UI

A frontend project inspired by platforms like Josh, Instagram Reels, and YouTube Shorts.

## 🚀 Features
- Scroll-based video autoplay
- Double tap ❤️ like animation
- Persistent likes using localStorage
- Comment toggle UI
- Tap to mute/unmute videos
- Responsive mobile-first design

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
